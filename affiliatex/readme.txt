#### Know exactly how your affiliate links are performing.

The new **AffiliateX Analytics** dashboard gives you a complete picture of your affiliate performance, right inside your WordPress admin. No third-party tools, no extra setup.

* **Click tracking** for every affiliate link on your site
* **Time range filtering** – view performance for today, last 7 days, 30 days, or custom ranges

Analytics is **100% free** and helps you make data-driven decisions to grow your affiliate income.

### Broken Links Checker (Free Feature)

#### Never lose a sale to a dead affiliate link again.

Affiliate links break. Products get discontinued, URLs change, and merchants update their structures. The new **Broken Links Checker** automatically finds dead affiliate links across your entire site so you can fix them before they cost you commissions.

* Detect broken affiliate links across all posts and pages
* See exactly which post contains each broken link
* Fix issues quickly from a single dashboard

Pro users also get **automatic scheduled scans** and **email alerts** the moment a link breaks. *(Available in [AffiliateX Pro](https://affiliatexblocks.com/pricing/?utm_source=wporg&utm_medium=affiliatex&utm_campaign=broken-links).)*

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select affiliatex.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download affiliatex.zip
2. Extract the affiliatex directory to your computer
3. Upload the affiliatex directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard

== Frequently Asked Questions ==

= Who is AffiliateX built for? =

AffiliateX is one of the best WordPress plugins for Amazon affiliates who want to create speed-optimized affiliate blogs with high Click-Through Rate (CTR) and conversion. It also supports eBay and AliExpress affiliates in the Pro version.

= Do I need an Amazon API key to use AffiliateX? =

No. While Amazon's Product Advertising API gives you the most automated experience, AffiliateX Pro includes a **Zero API** mode that lets you add Amazon products without API access – perfect for new affiliates who haven't yet qualified for the Creator API.

= Does AffiliateX track my affiliate clicks? =

Yes. The free **Analytics** feature gives you a full click-tracking dashboard with breakdowns by time range, referral source, block type, and page – all inside WordPress.

= How does the Broken Links Checker work? =

It automatically scans your site for affiliate links that no longer work and reports them in your AffiliateX dashboard so you can fix them quickly. Pro users also get scheduled auto-scans and email alerts.

= Can I use AffiliateX with eBay and AliExpress? =

Yes. AffiliateX Pro supports eBay Partner Network and AliExpress Affiliate Program in addition to Amazon. You can mix products from any of these networks within the same blocks.

= Does this plugin work with Classic Editor? =

No. AffiliateX works with Gutenberg and Elementor only.
